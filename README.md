# Pink Note Bug Fix

This mod for UNBEATABLE attempts to fix the "pink note bugs" in the game. Due to this changing how the game works, saving and submitting of scores will be disabled when this mod is installed.

## Compatible game versions

- UNBEATABLE (tested with `v1.7.1`)

## Requirements

- [BepInEx](https://github.com/BepInEx/BepInEx)

## Installation

1. Download and install BepInEx into your game directory (if you use [CustomBeatmaps](https://github.com/unbeatable-modding/CustomBeatmapsV5), you have this installed already)
2. Run the game, then close it
3. [Download this mod](https://github.com/Zachava96/PinkNoteBugFix/releases)
4. Merge the BepInEx folder from this mod with the BepInEx folder in your game directory
5. Run the game
